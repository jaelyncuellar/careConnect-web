  Run `npm i` to install the dependencies.

# Healthcare Management Platform

A healthcare operations platform designed for group homes and direct support professionals (DSPs). The application streamlines client care, staff management, behavioral documentation, scheduling, and operational reporting through role-based access controls.

## Features

* Secure authentication with separate Admin and Staff experiences
* Client management with detailed care profiles
* Behavioral support plans and observation tracking
* Medication and appointment management
* Staff scheduling and timesheet submission
* Group home occupancy and resident management
* Analytics dashboards and operational reporting

## User Roles

### Admin

* Organization-wide visibility across clients, staff, and group homes
* Reporting and analytics dashboards
* Staff management and operational oversight

### Staff

* Personalized dashboard
* Assigned client and group home access
* Schedule and timesheet management
* Observation and care documentation

## Client Profiles

Each client record includes:

* Overview
* Behavioral Plan
* Medications
* Appointments
* Observations

## Tech Stack

* React
* TypeScript
* Tailwind CSS
* Chart.js / Recharts
* Role-Based Access Control (RBAC)

## Purpose

Built to demonstrate modern healthcare administration workflows, behavioral support documentation, and staff coordination in a responsive, user-friendly application.
